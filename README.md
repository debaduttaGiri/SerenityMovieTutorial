# Serenity Movie Tutorial

Movie management web application built using **Serenity MVC (3.x)** and **ASP.NET Framework 4.8**.

## Modules Implemented
- Movie
- Genre
- Person
- MovieCast
- MovieGenres

These modules were generated using Serenity Code Generator and provide basic CRUD functionality.

## Tech Stack
- ASP.NET Framework 4.8
- Serenity MVC
- C#
- SQL Server
- Visual Studio 2019
